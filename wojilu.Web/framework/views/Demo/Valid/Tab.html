设置当前tab
给当前tab加上名为 currentTab 的 class。
    
            <li class="currentTab"><a href="/Demo/Valid/Tab.aspx">选项卡1</a><span></span></li>
自动设置当前tab(js代码)
如下js代码不是必须的。它只是帮你自动设置当前tab，你完全可以通过上面的 currentTab 手动设置
<script>
_run( function() {
    wojilu.tool.makeTab('.tabList', 'currentTab', '');
});
</script>
